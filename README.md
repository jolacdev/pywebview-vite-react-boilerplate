# PyWebView with Vite React

Simple boilerplate for _PyWebView_, and _Vite React_.

Stack based on PyWebView, Vite React, PyInstaller (Windows/Linux) and Py2App (macOS).

This is the [original boiler plate](https://github.com/r0x0r/pywebview-react-boilerplate).

## Requirements

- Python 3
- Node
- Pnpm

## Initialization

```shell
pnpm initialize
```

## Dev

### Only Frontend

```shell
pnpm dev
```

### Frontend + Backend

```shell
pnpm start
```

## Build

```shell
pnpm build
```

## Clean

```shell
pnpm clean
```
